MotionCraft · AI Video Orchestration

Unified control for multi-model video generation

Coordinate projects, shots, and render jobs across Pika, Runway, Luma, and Kling from a single hub. MotionCraft gives you a consistent control plane instead of juggling siloed tools.

Quick start

1) Clone & explore

  • Review domain models in server/src/core/models.ts.
  • See orchestration logic in server/src/core/engine.ts.

2) Configure providers

  • Copy .env.example → .env and add Pika/Runway/Luma/Kling keys.
  • Align payloads with the adapter stubs in server/src/providers.

3) Shape the UI

  • Edit the React shell at web/src/pages/index.tsx.
  • Style components via web/src/styles/base.css.

What you get

Unified schema

Projects, shots, and render jobs share one contract, so providers stay interchangeable.
