# Prompt Templates

Prompt templates help standardize style and pacing across shots. They can be expanded with shot-specific variables before submitting to providers.

## Cinematic Trailer
```
[SCENE] in ultra-wide frame, cinematic lighting, subtle camera movement.
Mood: [MOOD]. Color palette: [PALETTE].
Use dramatic depth of field and smooth transitions.
```

## Product Spotlight
```
Hero close-up of [PRODUCT], rotating on a clean background.
Studio lighting, crisp reflections, macro detail.
Include animated callouts for features: [FEATURES].
```

## Music Visualizer
```
Audio-reactive visuals matching BPM [BPM].
Neon strokes, volumetric fog, and particle bursts synced to beats.
Primary color: [COLOR].
```

## Tips
- Keep prompts concise; providers often truncate long text.
- Pair each template with reference images for more consistent results.
- Test each template with multiple providers to discover strengths and limitations.
